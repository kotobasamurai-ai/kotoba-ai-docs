共通エージェントは、複数のエージェントから `呼出` や `遷移` で再利用できるサブフローです。認証処理や住所確認など、共通する会話パターンを一度作成すれば、どのエージェントからでも呼び出せます。

## 共通エージェント一覧

サイドバーの「共通エージェント」をクリックすると、一覧が表示されます。

| 操作 | 説明 |
|------|------|
| 検索 | エージェント名でフィルタリング |
| 新規作成 | テーブル上部の「新しい共通エージェントを作成...」をクリック |
| 複製 | 既存の共通エージェントをコピー |
| 削除 | 不要な共通エージェントを削除 |

## エージェントエディタ

共通エージェントの編集画面には **エージェントエディタ** タブと **基本設定** タブがあります。

### Kotobaエディタ

エージェントエディタタブでは、Kotobaエディタを使って会話フローを作成します。

#### スラッシュコマンド（`/`）

エディタ内で `/` を入力するとコマンドメニューが表示されます。

| コマンド | 説明 |
|---------|------|
| 指示 | AIへの指示を追加 |
| 待機 | ユーザーの回答待ち条件を追加 |
| 実行 | 外部ツールの実行を追加 |
| もし | 条件分岐（if）を追加 |
| それ以外 | 条件分岐（else）を追加 |
| 呼出 | 別の共通エージェントを呼び出し（完了後に戻る） |
| 遷移 | 別のエージェントへ遷移（戻らない） |
| 転送 | オペレーターへの電話転送 |
| 終了 | 通話の終了 |

#### メンションショートカット

| 記号 | 対象 | 説明 |
|------|------|------|
| `@` | ツール | 登録済みツールの候補から選択 |
| `#` | エージェント | 共通エージェントの候補から選択 |
| `$` | 変数 | 利用可能な変数の候補から選択 |

<Tip>
  コマンドやメンションは候補リストから選択するだけでなく、日本語で直接入力しても自動変換されます。例えば「待機」と入力すると自動的に待機コマンドに変換されます。
</Tip>

### 基本設定タブ

| 項目 | 説明 |
|------|------|
| 名前 | 共通エージェントの名前。他のフローから `呼出` / `遷移` で参照する際に使います |
| 説明 | 共通エージェントの用途の説明 |

#### デフォルト変数

共通エージェント内で使用する変数のデフォルト値を設定できます。呼び出し元のフローから `inputs` で値を上書きできます。

```yaml
# 呼び出し元のフロー
- 呼出: 認証フロー
  inputs:
    顧客ID: "{{ 顧客ID }}"
```

## テスト通話

編集画面の **テスト通話** ボタンをクリックすると、エディタの右側にWebRTC通話パネルが開きます。作成中のフローを実際の音声で確認できます。

<Info>
  テスト通話はローカル環境（localhost）では利用できません。デプロイされた環境でお試しください。
</Info>

## 公開とバージョン管理

| 操作 | 説明 |
|------|------|
| **公開** | 現在の編集内容をバージョン付きスナップショットとして公開します |
| **バージョン履歴** | ヘッダーのバージョン履歴ボタンから過去のバージョンを確認・復元できます |

公開するたびにバージョン番号が自動的に付与されます。復元するとフロー定義と変数の両方が過去の状態に戻ります。
