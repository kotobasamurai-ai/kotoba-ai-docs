title: 条件分岐

---

`もし`（if）と `それ以外`（else）を使って、変数の値やツールの結果に応じてフローを分岐できます。

## 基本構文

```yaml
- if: "{{ 用件 }} == 技術"
  then:
    - 指示: "技術サポートへおつなぎします"
- else:
  then:
    - 指示: "一般窓口で対応します"
```

`if` の条件が真なら `then` ブロックが実行され、偽なら `else` の `then` ブロックが実行されます。

## 比較演算子

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `==` | 等しい | `{{ 用件 }} == 技術` |
| `!=` | 等しくない | `{{ 回答 }} != はい` |
| `<` | より小さい | `{{ @時刻取得.hour }} < 6` |
| `<=` | 以下 | `{{ @時刻取得.hour }} <= 9` |
| `>` | より大きい | `{{ @時刻取得.hour }} > 20` |
| `>=` | 以上 | `{{ @時刻取得.hour }} >= 18` |
| `in` | リストに含まれる | `{{ 部署 }} in [営業, 技術]` |

## 複数条件の分岐（if-if-else）

`if` を連続して並べると、上から順に評価され、最初に一致した条件が実行されます。
どの `if` にも一致しなかった場合は `else` が実行されます。

```yaml
- 指示: "用件を聞く（技術、料金、その他のいずれか）"
- 待機: "用件を聞き取るまで"
  saves:
    - 用件:
        type: string
        description: "お客様の用件"
        enum: ["技術", "料金", "その他"]

- if: "{{ 用件 }} == 技術"
  then:
    - 指示: "技術サポートへつなぐ旨を伝える"
- if: "{{ 用件 }} == 料金"
  then:
    - 指示: "料金サポートへつなぐ旨を伝える"
- else:
  then:
    - 指示: "一般対応する旨を伝える"
```

## in 演算子

値が複数の候補のいずれかに一致するかを判定します。

```yaml
- if: "{{ 部署 }} in [営業, 技術]"
  then:
    - 指示: "フロントオフィス部門の担当者へおつなぎします"
- if: "{{ 部署 }} in [総務, 人事]"
  then:
    - 指示: "バックオフィス部門の担当者へおつなぎします"
- else:
  then:
    - 指示: "一般窓口で対応します"
```

## 分岐後の合流

`if/else` の各ブロック内に `終了` や `遷移` がなければ、分岐後にメインフローへ戻ります。

```yaml
- if: "{{ 用件 }} == 技術"
  then:
    - 指示: "技術的な解決策を提案する"
    - 待機: "ユーザーが解決策を承認するまで"
- else:
  then:
    - 指示: "一般的な解決策を提案する"
    - 待機: "ユーザーが解決策を承認するまで"

# ここに合流する
- 指示: "他にご質問はありますか？"
```

## グローバル変数との組み合わせ

時刻や曜日などのグローバル変数を使って、営業時間外の判定ができます。

```yaml
- if: "{{ global.時刻.時 }} >= 18"
  then:
    - 指示: "営業時間外であることを伝える"
    - 終了: true
- if: "{{ global.時刻.時 }} < 9"
  then:
    - 指示: "営業時間外であることを伝える"
    - 終了: true

- if: "{{ global.時刻.曜日 }} in [土, 日]"
  then:
    - 指示: "本日は休業日であることを伝える"
    - 終了: true
```

### 利用可能なグローバル変数

| 変数 | 型 | 例 | 説明 |
|------|-----|-----|------|
| `global.時刻.年` | number | 2026 | 西暦年 |
| `global.時刻.月` | number | 1 | 月（1-12） |
| `global.時刻.日` | number | 22 | 日（1-31） |
| `global.時刻.時` | number | 15 | 時（0-23） |
| `global.時刻.分` | number | 30 | 分（0-59） |
| `global.時刻.曜日` | string | 水 | 曜日（月〜日） |
| `global.時刻.日付` | string | 2026-01-22 | 日付 |
| `global.時刻.時刻` | string | 15:30 | 時刻 |

## ツール結果での分岐

ツール実行後、その結果を条件に分岐できます。

```yaml
- 指示: "顧客IDを聞く"
  saves:
    - 顧客ID: { type: string }
- 実行: "@customer_lookup"
  inputs:
    id: "{{ 顧客ID }}"
- if: "{{ @customer_lookup.success }} == False"
  then:
    - 指示: "顧客が見つからないことを伝える"
    - 終了: true
- else:
  then:
    - 指示: "{{ @customer_lookup.name }}さんに挨拶する"
```
