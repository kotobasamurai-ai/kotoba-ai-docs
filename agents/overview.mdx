エージェントは、AIが電話でどのように会話するかを定義する設定です。
**base_prompt**（AIの役割や基本方針）と **steps**（会話の進行手順）の2つで構成されます。

## 最小構成の例

```yaml
base_prompt: |
  あなたはカスタマーサポートのAIオペレーターです。
  丁寧な対応を心がけてください。

steps:
  - 指示: "挨拶をして、お客様のお名前をフルネームで聞く"
  - 待機: "お名前を聞き取れるまで"
    saves:
      - 氏名:
          type: string
          description: "お客様の氏名"
  - 指示: "{{ 氏名 }}様、お電話ありがとうございました。失礼いたします。"
  - 終了: true
```

AIが挨拶して名前を聞き、名前を取得したら終話するシンプルなフローです。

## 構成要素

| 要素 | 説明 |
|------|------|
| `base_prompt` | AIの役割・方針を定義するシステムプロンプト |
| `指示` | AIへの指示。何を話すか、何を聞くかを自然言語で記述 |
| `待機` | 次に進む条件。ユーザーの回答が条件を満たすまで会話を続ける |
| `saves` | ユーザーから収集する情報の定義 |
| `もし` / `それ以外` | 条件分岐。変数の値やユーザーの回答に応じてフローを分ける |
| `実行` | 外部ツールの実行 |
| `呼出` | サブフローの呼び出し（完了後に戻る） |
| `遷移` | 別フローへの遷移（戻らない） |
| `転送` | オペレーターへの電話転送 |
| `終了` | 通話の終了 |
