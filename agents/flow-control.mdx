複数のフローを組み合わせたり、オペレーターへ電話を転送する方法を説明します。

## 呼出（call）— サブフローを呼び出して戻る

`呼出` は別のフローを呼び出し、完了後にメインフローへ戻ります。
認証や住所確認など、複数のエージェントで共通する処理を再利用できます。

```yaml
- 呼出: 認証フロー
  inputs:
    顧客ID: "{{ 顧客ID }}"
```

### サブフローの変数を参照する

呼び出したフロー内で保存された変数は `{{ #フロー名.変数名 }}` で参照できます。

```yaml
- 呼出: 認証フロー
  inputs:
    顧客ID: "{{ 顧客ID }}"

# 認証フロー内で保存された変数を参照
- 指示: "認証結果は {{ #認証フロー.ステータス }} です"
```

サブフロー内で実行したツールの結果も参照できます。

```yaml
- 指示: "{{ #認証フロー.@customer_lookup.name }}様、ご本人確認ができました"
```

## 遷移（goto）— 別フローへ移動する（戻らない）

`遷移` は現在のフローを終了し、別のフローへ一方向に移動します。`呼出` と異なり、元のフローには戻りません。

```yaml
- if: "{{ 用件 }} == 技術"
  then:
    - 指示: "技術サポートへおつなぎします"
    - 遷移: 技術サポートフロー
- else:
  then:
    - 指示: "一般窓口で対応します"
    - 遷移: 一般対応フロー
```

### 呼出と遷移の使い分け

| | 呼出（call） | 遷移（goto） |
|---|---|---|
| **完了後** | 元のフローに戻る | 戻らない |
| **用途** | 共通処理の再利用 | フローの切り替え |
| **例** | 認証、住所確認 | 部門別対応への振り分け |

## 転送（transfer）— オペレーターへ電話を転送する

`転送` はAIからオペレーター（人間）へ電話を転送します。

### コールド転送

AIが即座に電話を転送します。

```yaml
- 指示: "担当者へおつなぎします"
- 転送:
    to: "+818012345678"
    type: cold
    message: "担当者におつなぎいたします。"
```

### ウォーム転送

AIがまずオペレーターに用件を引き継いでから、お客様をつなぎます。

```yaml
- 指示: "担当者へおつなぎします"
- 転送:
    to: "{{ 転送先番号 }}"
    type: warm
    message: "{{ 用件詳細 }}の件で、担当者におつなぎいたします。少々お待ちください。"
    briefing: auto
```

### 転送のプロパティ

| プロパティ | 必須 | 説明 |
|-----------|------|------|
| `to` | はい | 転送先の電話番号（E.164形式）またはSIP URI |
| `type` | いいえ | `cold`（デフォルト）または `warm` |
| `message` | いいえ | お客様への転送メッセージ |
| `briefing` | いいえ | オペレーターへの引き継ぎ内容。`auto` でAIが自動生成 |
| `timeout` | いいえ | オペレーターの応答待ち時間（秒、デフォルト: 30） |

<Info>
  `to` には変数も使えます（例: `{{ 転送先番号 }}`）。ユーザーに転送先を聞いて動的に転送できます。
</Info>

## 実践例

用件に応じて、技術サポートフローへ遷移するか、オペレーターへ転送する例です。

```yaml
base_prompt: |
  あなたはカスタマーサポートのAIオペレーターです。

steps:
  - 指示: "用件と緊急度を聞く"
  - 待機: "用件と緊急度を聞き取るまで"
    saves:
      - 用件:
          type: string
          description: "お客様の用件"
          enum: ["技術", "料金", "その他"]
      - 緊急度:
          type: string
          description: "緊急度"
          enum: ["高", "中", "低"]

  - if: "{{ 用件 }} == 技術"
    then:
      - 指示: "技術サポートへおつなぎします"
      - 遷移: 技術サポートフロー

  - if: "{{ 緊急度 }} == 高"
    then:
      - 指示: "担当者へ緊急転送します"
      - 転送:
          to: "+818012345678"
          type: warm
          message: "緊急のお問い合わせです。担当者におつなぎいたします。"
          briefing: auto
  - else:
    then:
      - 呼出: 一般対応フロー
        inputs:
          用件: "{{ 用件 }}"
      - 指示: "対応が完了しました。他にご質問はありますか？"
      - 終了: true
```
