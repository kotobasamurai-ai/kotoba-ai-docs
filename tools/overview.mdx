ツールは、エージェントが通話中または通話終了後に実行する処理です。SMS送信、通話終了、情報抽出、外部システム連携などをツールとして追加できます。

ツールは**標準ツール**と**カスタムツール**の2種類があります。標準ツールはスイッチで有効化できる組み込み機能で、カスタムツールは Python で独自ロジックを実装します。

## ツールの種類

<CardGroup cols={2}>
  <Card title="標準ツール" icon="shapes" href="/tools/realtime-tools">
    プラットフォームが提供する組み込みツールです。通話中（リアルタイム）と通話後の2種類があります。
  </Card>
  <Card title="カスタムツール" icon="code" href="/tools/custom-tools">
    Python で作成する独自ツールです。業務固有の処理を実装し、リアルタイム・通話後の両方で利用できます。
  </Card>
</CardGroup>

## 使い分けの目安

| 目的 | 選ぶもの |
| --- | --- |
| 会話中に即座に実行したい | 標準ツール（リアルタイム） |
| 通話後に自動でまとめて処理したい | 標準ツール（通話後） |
| 自社独自の業務ロジックを実装したい | カスタムツール |
