---

ツールは、エージェントが通話中または通話終了後に実行する処理です。SMS送信、通話終了、情報抽出、外部システム連携などをツールとして追加できます。

まずは「通話中に必要な処理か」「通話後にまとめて実行すればよい処理か」で分類すると整理しやすくなります。

## ツールの種類

<CardGroup cols={3}>
  <Card title="リアルタイムツール" icon="bolt" href="/tools/realtime-tools">
    通話中にエージェントが呼び出すツールです。通話終了、転送、SMS送信など、その場で実行したい処理を設定します。
  </Card>
  <Card title="通話後ツール" icon="clock" href="/tools/post-call-tools">
    通話終了後に自動実行されるツールです。要約生成や情報抽出など、後処理に向いた処理を設定します。
  </Card>
  <Card title="カスタムツール" icon="code" href="/tools/custom-tools">
    Python で作成する独自ツールです。業務固有の処理を実装し、リアルタイム・通話後の両方で利用できます。
  </Card>
</CardGroup>

## 使い分けの目安

| 目的 | 選ぶもの |
| --- | --- |
| 会話中に即座に実行したい | リアルタイムツール |
| 通話後に自動でまとめて処理したい | 通話後ツール |
| 自社独自の業務ロジックを実装したい | カスタムツール |
