ボイスクローンは、録音した音声やアップロードした音声ファイルをもとに、エージェントが話すためのカスタム音声を作成する機能です。作成した音声は、エージェント設定画面の「音声」で通常のプリセット音声と同じように選択できます。

## 作成の流れ

1. サイドメニューから「ボイスクローン」を開く
2. 右上の「新規作成」をクリックする
3. クローン音声の名前を入力する
4. 音声を録音する、またはファイルをアップロードする
5. 「作成する」をクリックして生成を開始する
6. ステータスが「完了」になったら、エージェントの音声として利用する

## 入力項目

### 名前

管理用の名称です。あとで一覧から見分けやすい名前を付けてください。

**例**: `受付女性音声`、`田中さん案内音声`

### 音声

音声の登録方法は2種類あります。

| 方法 | 説明 |
| --- | --- |
| 録音する | その場でマイクを使って録音します |
| ファイルを選択 | 既存の音声ファイルをアップロードします |

## 録音して作成する

マイクへのアクセスを許可すると録音を開始できます。

| 項目 | 内容 |
| --- | --- |
| 録音時間 | 10秒〜3分30秒 |
| 最低録音時間 | 10秒未満では停止できません |
| 録音後の確認 | その場で再生して確認できます |
| 録り直し | 「やり直す」で再録音できます |

<Tip>
  収録時間の長さよりも、雑音の少なさと話者の安定性の方が品質に影響します。短くても明瞭な音声を優先してください。
</Tip>

## ファイルをアップロードして作成する

既存の音声ファイルからクローンを作成することもできます。

| 項目 | 内容 |
| --- | --- |
| 対応形式 | `.wav`, `.mp3`, `.ogg`, `.webm`, `.mp4`, `.m4a` |
| アップロード後の確認 | その場で再生して確認できます |
| 差し替え | 「やり直す」で別のファイルに変更できます |

## ステータス

作成したボイスクローンは一覧に表示され、状態が更新されます。待機中や作成中のクローンは、画面上で自動的に状態が反映されます。

| ステータス | 説明 |
| --- | --- |
| 待機中 | 作成処理の開始待ちです |
| 作成中 | 音声クローンを生成しています |
| 完了 | 利用可能です。エージェント設定画面から選択できます |
| 失敗 | 作成に失敗しています。エラーメッセージが表示されます |

## 作成後の使い方

クローンが完了すると、一覧に `fish/{voice_id}` が表示されます。このIDはコピーできますが、通常は手入力する必要はありません。

エージェント設定画面の「音声」で Fish Audio を選択すると、ボイス一覧の中に「クローン」として表示され、そこから選択できます。

<Info>
  実際の聞こえ方は会話内容やTTS設定でも変わります。クローン作成後は、本番利用の前にテスト通話で確認してください。
</Info>

## 削除

不要になったボイスクローンは一覧から削除できます。削除は取り消せません。

<Warning>
  利用中のエージェントがある場合は、削除前に他のボイスに変更してください。
</Warning>

## おすすめの音声素材

より安定したボイスクローンを作るには、次のような音声が推奨されます。

- 周囲の雑音が少ない
- 話者が1人だけである
- 声がはっきりしている
- 早口すぎず、抑揚が極端すぎない
- できるだけ元音声に近いクリアな録音である
