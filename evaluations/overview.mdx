評価機能では、エージェントの通話品質をメトリクスに基づいて自動評価できます。対象期間の通話ログに対してAIが評価を実行し、スコアと推論理由を確認できます。

## 評価の作成

<Steps>
  <Step title="エージェントを選択">
    評価対象のエージェントをドロップダウンから選択します。ルーティングされた電話番号がある場合は番号も表示されます。
  </Step>

  <Step title="対象期間を選択">
    評価する通話の期間を日付ピッカーで指定します。デフォルトは過去7日間です。
  </Step>

  <Step title="メトリクスを選択">
    評価に使用するメトリクスを選択します。

    | メトリクス | 説明 |
    |-----------|------|
    | **有人転送率** | エスカレーション（有人転送）なしで解決できた通話の割合を評価 |
    | **顧客満足度** | 通話内容から顧客の満足度を推定 |
    | **カスタムプロンプト** | 独自の評価基準をプロンプトとして記述 |

    有人転送率と顧客満足度を選択すると、プリセットの評価プロンプトが自動入力されます。カスタムプロンプトを選択した場合は、自由に評価基準を記述できます。
  </Step>

  <Step title="評価プロンプトを確認・編集">
    評価に使用するプロンプトを確認し、必要に応じて編集します。プロンプト内の `{{transcript}}` は実際の通話トランスクリプトに置き換えられます。
  </Step>

  <Step title="評価を実行">
    **実行** ボタンをクリックすると、対象期間の通話に対して評価が開始されます。処理状況はリアルタイムで更新されます。
  </Step>
</Steps>

## 評価結果の確認

画面は左右2つのパネルで構成されています。

### 左パネル：評価履歴

過去に実行した評価の一覧が表示されます。各項目には以下の情報が表示されます：

- エージェント名
- 実行日時
- メトリクス種別
- 処理状況（処理済み / 総セッション数）
- 平均スコア

### 右パネル：評価詳細

評価履歴から項目を選択すると、詳細が表示されます。

| 項目 | 説明 |
|------|------|
| ステータス | `queued`（待機中）/ `in_progress`（実行中）/ `completed`（完了） |
| メトリクス | 使用した評価メトリクス |
| 平均スコア | 全セッションの平均評価スコア |

#### 結果テーブル

各通話セッションの評価結果が一覧で表示されます。

| 列 | 説明 |
|-----|------|
| 日時 / セッションID | 通話の日時とセッション識別子 |
| スコア | 評価スコア（色分け表示） |
| AIによる評価理由 | AIが判定した評価の根拠 |

結果行をクリックすると、サイドパネルが開き、セッションIDやスコアに加えて通話のトランスクリプト全文を確認できます。

## CSVエクスポート

評価結果はCSV形式でエクスポートできます。評価詳細画面のエクスポートボタンからダウンロードできます。
