通話ログでは、エージェントが対応した全通話の履歴を確認できます。通話中のセッションはリアルタイムで更新されます。

## 通話ログ一覧

サイドバーの「通話ログ」をクリックすると、通話履歴がテーブル形式で表示されます。

| 列 | 説明 |
|-----|------|
| ステータス | `進行中`（通話中）または `完了` |
| 要約 | AIが生成した通話サマリー（冒頭部分を表示） |
| 電話番号 | 発信元の電話番号 |
| 開始時刻 | 通話開始日時（MM/DD HH:MM） |
| 通話時間 | 通話の長さ（M:SS 形式） |

<Info>
  通話ログはリアルタイムで更新されます。新しい通話が開始されると自動的に一覧に追加され、ステータスが「進行中」として表示されます。
</Info>

## セッション詳細

一覧から通話をクリックすると、詳細画面が開きます。画面は左右2つのパネルで構成されています。

### 左パネル：サマリー

| 項目 | 説明 |
|------|------|
| ステータス | 進行中 / 完了 |
| セッションID | 通話の一意な識別子 |
| AI サマリー | AIが生成した通話内容の要約 |
| 開始時刻 | 通話開始日時 |
| 終了時刻 | 通話終了日時 |
| 通話時間 | 通話の長さ（M分SS秒） |
| 最終更新 | 最後にデータが更新された日時 |

<Tip>
  AIサマリーは通話完了後に自動生成されます。進行中の通話ではまだ表示されません。
</Tip>

### 右パネル：会話ログと録音

#### 会話ログ

チャットバブル形式で通話のトランスクリプト（文字起こし）が表示されます。AIの発話とユーザーの発話が左右に分かれて表示されるため、会話の流れを直感的に確認できます。

#### 録音再生

通話が完了したセッションでは、会話ログの下に録音プレーヤーが表示されます。通話の音声を再生して確認できます。
