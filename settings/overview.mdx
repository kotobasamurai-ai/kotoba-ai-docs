設定ページでは、自分のプロフィール情報、組織メンバー、月ごとの利用状況など、運用に必要な管理情報を確認できます。

## プロフィール

プロフィールでは、現在ログインしているユーザーと所属組織の基本情報を確認できます。

| 項目 | 説明 |
| --- | --- |
| メールアドレス | 現在ログインしているユーザーのメールアドレスが表示されます |
| 組織名 | 所属している組織名が表示されます。未設定の場合は `未設定` と表示されます |

<Info>
  パスワード等の変更が必要な場合は、小幡(obata@kotobasamurai.jp)にお問い合わせください。
</Info>

## メンバー

メンバーでは、組織に所属しているユーザーの一覧を確認し、招待や削除を行えます。

一覧には次の情報が表示されます。

| 項目 | 説明 |
| --- | --- |
| メールアドレス | メンバーのログインメールアドレス |
| 権限 | 組織内での権限 |
| 参加日 | 組織に参加した日 |

### メンバーを招待する

招待したいユーザーのメールアドレスを入力して、組織へ追加できます。

### メンバーを削除する

自分以外のメンバーは削除できます。

<Warning>
  削除すると、そのユーザーは組織内の設定やデータにアクセスできなくなります。
</Warning>

## 使用量

使用量ページでは、月ごとの通話実績と推定料金を確認できます。

| 項目 | 説明 |
| --- | --- |
| 通話数 | その月に発生した通話件数 |
| 合計通話時間 | その月の総通話時間 |
| 推定料金 | 契約プランに応じて計算された推定料金 |

直近12か月分を月単位で切り替えて確認できます。

### プラン情報

料金プランが設定されている場合、以下の情報も表示されます。

| 項目 | 説明 |
| --- | --- |
| プラン名 | 契約中のプラン |
| 基本料金 | 月額の基本料金 |
| 通話料単価 | 1分あたりの通話料金 |
| 無料通話枠 | プランに含まれる無料枠 |
| 課金対象分数 | 無料枠を超えた分数 |

<Info>
  料金プランが未設定の場合、推定料金は表示されません。
</Info>
